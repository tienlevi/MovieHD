# MovieHD

Free online movie website 

# Official Link

[https://movie-hd.vercel.app/](https://movie-hd.vercel.app/)

# Technology

- React
- SCSS
- Firebase

# Feature

- Watch video movie
- Search name movie or type movie
- Sign In with Google or Facebook
- Comment
- Add favorite movie
- Dark Light Mode
